# Greentany Consulting

## Project info

Ce projet est un site vitrine pour Greentany Consulting, expert en conformité, qualité, sécurité alimentaire et normes sociales.

## Installation et développement

Prérequis : Node.js & npm installés ([voir nvm](https://github.com/nvm-sh/nvm#installing-and-updating))

```sh
# Cloner le dépôt
git clone <YOUR_GIT_URL>

# Aller dans le dossier du projet
cd <YOUR_PROJECT_NAME>

# Installer les dépendances
npm i

# Lancer le serveur de développement
npm run dev
```

## Technologies utilisées

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Déploiement

Déployer sur la plateforme de votre choix (Vercel, Netlify, etc.) ou sur un serveur web statique.

## Personnalisation du domaine

Pour connecter un domaine personnalisé, suivez la documentation de votre hébergeur (Vercel, Netlify, etc.).
